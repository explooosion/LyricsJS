# LyricsJS
a dynamic lyrics pachage

## Install
```bash
git clone https://github.com/explooosion/LyricsJS.git
```  
```bash
npm install
```  
  
+ run with nodemon
```bash
npm install nodemon -g
npm run dev
```  
+ run with pm2
```bash
npm install pm2 -g
npm start
npm stop
```
